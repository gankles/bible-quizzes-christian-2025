Create the **Homepage** for the Bible Quiz website.

**DESIGN SYSTEM (REQUIRED):**

## Visual Theme & Atmosphere
- Scholarly yet accessible, reverent but modern
- Clean, content-focused, with subtle biblical theming
- Medium density - balanced whitespace with rich content areas

## Color Palette & Roles
- **Deep Navy (#2C3E50):** Primary text, headers
- **Royal Blue (#3B82F6):** Primary actions, links
- **Warm Gold (#C9A961):** Accents, highlights
- **Soft Cream (#F5F1E8):** Alternative backgrounds
- **Light Gray (#F8FAF9):** Page backgrounds

## Typography Rules
- **Headings:** System sans-serif with tight tracking
- **Body:** System sans-serif, line-height 1.6-1.8
- **Scripture:** Serif font, italic style
- **H1:** 48-72px, font-black, navy color
- **Labels:** All caps, wide tracking (0.1em), small size

## Component Stylings
- **Primary Button:** bg-blue-600, white text, rounded-lg, hover:bg-blue-700
- **Cards:** bg-white, rounded-xl, border border-slate-200, shadow-sm
- **Hover State:** hover:shadow-md hover:border-blue-300
- **Badges:** px-3 py-1, rounded-full, bg-blue-50 text-blue-600

## Layout Principles
- **Max-width:** max-w-7xl (1280px)
- **Section padding:** py-16 to py-24
- **Grid:** Responsive grids with md:grid-cols-2 lg:grid-cols-4
