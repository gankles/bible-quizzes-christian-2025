## Usage Instructions

1. Go to stitch.withgoogle.com
2. Create new project or use existing
3. Click "Generate" or "Add Screen"
4. Paste one prompt at a time
5. Wait for generation (may take 1-2 minutes per screen)
6. Download HTML/CSS or iterate

**Tip:** If a prompt times out, try splitting it into smaller sections or simplify the description slightly.
